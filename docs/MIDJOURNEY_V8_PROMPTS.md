## V8 PROMPTING STRATEGY

V8 Alpha shifts the game. Key differences from V7:
- **Longer, more detailed prompts** perform dramatically better
- **`--raw`** removes default styling for maximum prompt control — use this for backgrounds where we need specific color/composition
- **`--hd`** generates native 2K without upscaling artifacts — use for hero images and print
- **`--s 1000`** cranks stylization to max — V8 shines here, especially with personalization
- **`--chaos`** and **`--weird`** still supported for happy accidents
- **Quoted text** now renders legibly — useful for any image with embedded typography
- **5x faster** than V7 at standard res — explore cheap, upscale winners

### Workflow
1. Generate at standard res first (fast, cheap)
2. Browse in Grid Mode
3. Upscale winners with `--hd` for native 2K
4. Animate top picks via Kling AI or Grok Imagine

## GENERATION STRATEGY

### Round 1: Explore (Standard Res, Fast)
Generate all prompts at standard resolution. Cost: 1x GPU per image.
- Run H1-H4 (heroes)
- Run A1-A6 (backgrounds)
- Run G1-G4 (Ghibli fusion)
- Run D1-D3 (dividers)
- Run C1-C2 (contact)
Total: ~19 generations at standard = ~19 GPU minutes

### Round 2: Select & Upscale
Pick the best variation of each, upscale with `--hd` for native 2K.
Cost: 4x per upscale.

### Round 3: Animate
Send top 5-7 hero/section images to Kling AI for 5-10s cinematic video clips.

### Round 4: Variations
For any image that's close but not perfect, use V8's variation tools to iterate.
Use `--chaos 30` on a few to get unexpected magic.
